# office-yysports-lab

这是一个用于归档和发布多个独立 HTML 页面的仓库。  
本仓库不针对任何特定域名或网站，仅作为静态 HTML 资源的存放与分发点。

## 项目简介

`office-yysports-lab` 主要用于整理和存放各类独立开发的 HTML 页面。  
这些页面可能包含实验性功能、工具页面、演示页面或其他前端资源。  
仓库本身不承载任何动态逻辑，所有内容均为静态 HTML。

## 目录结构

```
.
├── pages/            # 存放独立的 HTML 页面文件
├── assets/           # 公共资源（如样式、脚本、图片等）
├── README.md         # 本文件
└── index.html        # 可选：仓库根目录入口页
```

目录结构可能会随着归档内容的增加而调整。  
建议将每个独立页面放置于 `pages/` 下，并保持命名清晰。

## 页面归档说明

- 每个 HTML 页面均为独立文件，可直接在浏览器中打开运行。
- 页面之间不存在相互依赖关系（除非特别说明）。
- 归档内容可能不定期更新、清理或重组。
- 本仓库不保证所有页面在未来的可用性或兼容性。

## 使用方式

你可以通过以下方式访问仓库中的 HTML 页面：

1. 直接克隆仓库到本地，用浏览器打开对应 HTML 文件。
2. 通过 GitHub Pages 功能发布（如已启用），访问对应 URL。
